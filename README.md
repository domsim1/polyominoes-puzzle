# Polyominoes Puzzle

## Buttons

|Key|Action|
|---|------|
| a | move left |
| d | move right |
| s | soft drop |
| w | hard drop |
| j | rotate left |
| k | rotate right |
| space | hold |


## How to build

Requires [raylib-go](https://github.com/gen2brain/raylib-go)

```
go run src/main.go
```
